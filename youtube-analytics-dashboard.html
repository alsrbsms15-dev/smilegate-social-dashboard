      ⚠️ 보안 권장사항

      키에는 YouTube Data API v3만 허용하도록 API 제한을 걸고, 가능하면 HTTP 리퍼러 제한(예: file://, 사용하시는 도메인)을 추가하세요. 노출된 키는 즉시 폐기/재발급하세요.

      로컬(file://)에서 여는 경우 HTTP 리퍼러 제한은 "없음"으로 두거나 IP 제한만 걸어야 작동합니다.
